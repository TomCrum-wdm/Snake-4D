# 贪食蛇4D (Snake 4D)

这是一个4D贪食蛇游戏项目。

This is a 4D Snake Game project.

## 安装 (Installation)

在浏览器中打开 `index.html` 文件即可运行游戏。

Open the `index.html` file in your browser to run the game.
